
# مشروع رؤيا - توليد تقارير PDF ذكية لبيانات الأصول

## الملفات المضمنة:
- `ro2ya_report.py`: سكربت بايثون لتوليد تقرير PDF
- `Logo-04.png`: شعار الجهة
- `Cairo-Regular.ttf`: خط يدعم العربية والإنجليزية
- `employee_assets.xlsx`: نموذج بيانات

## لتشغيل التقرير:
1. ثبّت المتطلبات:
   pip install reportlab arabic-reshaper python-bidi pandas

2. شغّل السكربت:
   python ro2ya_report.py

سيتم توليد الملف `SGS_Assets_Report_Bilingual.pdf` تلقائيًا.
